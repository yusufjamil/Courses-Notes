1. AWS Free Tier

Before creating an account, the instructor explains the AWS Free Tier.

What is the AWS Free Tier?

AWS provides free access to many services for new users.

It allows beginners to:

Learn AWS
Practice cloud concepts
Build small applications
Prepare for certifications

without paying for most basic usage.

Important

The Free Tier does not mean everything in AWS is free.

Only specific services and usage limits are included.

If those limits are exceeded, AWS starts charging automatically.

Important Interview Questions
What is the AWS Free Tier?

A program that provides limited free access to selected AWS services for new AWS customers, subject to usage limits.

Is AWS completely free?

No. Only eligible services and specified usage limits are free. Exceeding those limits or using non-eligible services results in charges.

Why does AWS ask for a credit card?

To verify the account and bill for any usage beyond the Free Tier or for paid services.

What is the Root User?

The account created during AWS registration that has full administrative access to all AWS resources and billing.

Why should you avoid using the Root User daily?

Because it has unrestricted permissions. AWS recommends creating IAM users for routine administration to reduce security risks.

Which support plan should beginners choose?

The Basic Support Plan, as it is free and adequate for learning.

Best Practices
Enable Multi-Factor Authentication (MFA) on the root account.
Use the root account only for account-level tasks.
Create an IAM administrator user for everyday management.
Monitor the AWS Billing Dashboard regularly.
Set up billing alerts to avoid unexpected charges.
Stay within Free Tier limits while learning.
