EFS vs EBS vs S3


| Feature      | EFS          | EBS                 | S3              |
| ------------ | ------------ | ------------------- | --------------- |
| Type         | File Storage | Block Storage       | Object Storage  |
| Mount        | Multiple EC2 | One EC2 (typically) | API Access      |
| Protocol     | NFS          | Block Device        | REST API        |
| Auto Scaling | Yes          | No                  | Yes             |
| Multi-AZ     | Yes          | No                  | Yes             |
| Shared       | Yes          | No                  | Yes             |
| Best For     | Shared Files | OS, Database        | Backups, Images |
